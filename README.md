### What's this?
Leveling bot for Discord. Meant to be flexible and rewarding 

## How to make it work on unix machines
```commandline
python3 -m venv venv
source venv/bin/activate
pip install pipenv
pipenv install
```

## Project setup

```bash
cp .env.example .env
```

### Run script locally
```commandline
python main.py
```
